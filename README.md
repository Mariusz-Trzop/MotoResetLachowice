# MotoReset Lachowice – Strona klubu motocyklowego

Statyczna strona klubu motocyklowego **MotoReset Lachowice**, hostowana na GitHub Pages.

## Podgląd online

Strona jest dostępna pod adresem:

https://mariusz-trzop.github.io/MotoResetLachowice/

## Technologie

- HTML5
- CSS3 (własny layout, responsywne menu, tryb zakładek)
- Prosty JavaScript (hamburger menu + przełączanie sekcji)

## Struktura

- `index.html` – główny plik strony
- `style.css` – style wyglądu
- (opcjonalnie) `img/` – zdjęcia do galerii
